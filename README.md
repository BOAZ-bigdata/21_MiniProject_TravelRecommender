## Project Summary
리뷰 기반 서울 지역 내 주요 관광 명소 추천 시스템 🌴 👍 <br/>
: 자신이 마음에 들어하는 관광지를 입력하면 그에 걸맞는 유사한 관광지들을 추천해준다. <br/>
: tripadvisor의 리뷰들을 크롤링하여 전처리하고 BERT / Doc2Vec을 이용하여 코사인 유사도를 측정, 상위 3~5개 유사한 관광지 추천<br/>

<br/>
